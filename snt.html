faire des mods sur Terraria (en C#)

pour faire des mods sur terraria la communauté a créé un mod pour faire des mods (tmodloader)

avec ce mod on peut faire

  • des objets personnalisés
  • des armes
  • des sceptres
  • des recettes
  • des musiques
  • et bien d'autres

tout cela se fait en C#

using Terraria;
using Terraria.ID;
 using Terraria.ModLoader;
 namespace ModNamespaceHere
 {
public class NameHere : ModItem
 {
public override void SetStaticDefaults()
 {
Tooltip.SetDefault("This is a modded item.");
}
public override void SetDefaults()
{
item.width = 20;
item.height = 20;
item.maxStack = 999;
item.value = 100;
item.rare = 1;
// Set other item.X values here
}
public override void AddRecipes()
{
// Recipes here. See Basic Recipe Guide
}
}
}
les mots public nous donne son niveau d'accès, void son type, override veut dire que l'on modifie la méthode
